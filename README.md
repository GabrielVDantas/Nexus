## Sobre
O mercado de jogos se mostra crescente a cada ano. Pensando nisso, o Nexus vem para auxiliar desenvolvedores indie para que possam competir com grandes empresas do ramo e acharem o seu espaço no mercado.

O desenvolvedor poderá postar seu projeto na plataforma e receber auxílio financeiro e feedback de outros usuários.
